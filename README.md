# Premium Dental Clinic Website

A stunning, modern, and fully responsive dentist website built with HTML, CSS, and JavaScript.

## Features

- Modern Hero Section with animated particles
- About Section with animated counters
- Services Grid with 3D tilt effect
- How It Works timeline
- Team Section with flip cards
- Auto-scrolling Testimonials
- Masonry Gallery with lightbox
- Appointment Booking Form
- WhatsApp Integration
- GSAP Animations
- Responsive Design
- Custom Cursor Effect

## Tech Stack

- HTML5
- CSS3 (Custom Animations)
- Vanilla JavaScript
- GSAP (Animations)
- Google Fonts
- Font Awesome Icons

## Colors

- Deep Navy: #0A2540
- Teal: #00C9A7
- Sky Blue: #E8F4FD
- White: #FFFFFF
